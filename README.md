## ğŸ“Š Dataset

**Boston Housing Dataset** â€” 506 samples, 13 input features, 1 target variable.

| Feature | Description |
|---------|-------------|
| `CRIM` | Per capita crime rate by town |
| `ZN` | Proportion of residential land (lots > 25,000 sq.ft.) |
| `INDUS` | Proportion of non-retail business acres |
| `CHAS` | Charles River boundary (1 = yes, 0 = no) |
| `NOX` | Nitric oxide concentration |
| `RM` | Average number of rooms per dwelling |
| `AGE` | Proportion of units built before 1940 |
| `DIS` | Distance to Boston employment centres |
| `RAD` | Highway accessibility index |
| `TAX` | Property-tax rate per $10,000 |
| `PTRATIO` | Pupil-teacher ratio |
| `B` | Measure of Black resident proportion |
| `LSTAT` | % lower-status population |
| **`MEDV`** | **Target: Median house value ($1000s)** |
